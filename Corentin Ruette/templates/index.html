Vote cantine

Ce projet a été réalisé en commun avec les étudiants des deux options. Ils ont collaboré pour construire un pupitre connecté en xbee avec un serveur afin de transmettre les votes des étudiants. Le serveur collecte les données dans une base de données et les diffuse au chef de cantine par un site web et aux utilisateurs de la cantine par un panneau LED de diffusion de message.

Technologie :

  • Electronique embarquée
  • Arduino
  • Xbee
  • Wifi
  • C++
  • QT/QML
  • HTML/CSS/PHP/SQL

Plots lumineux

Les étudiants ont réalisé des plots lumineux pour entrainer des sportifs. A partir d'un smartphone, les sportifs peuvent choisir un type d'entrainement (réflexe, endurance...) et ensuite les plots sont allumés pour indiquer au sportifs de le toucher. Le touché est détécté par le plot et ensuite le temps de réaction est calculé et transmis à un serveur pour faire des statistiques.

Technologie :

  • Electronique embarquée
  • Arduino
  • Xbee
  • Wifi
  • C++
  • QT/QML
  • HTML/CSS/PHP/SQL
